# Rainfall-dashboard-for-Nigeria-2014-to-2024
This repository contains the code used to create an interactive dashboard for analytics of rainfall from 2014 to 2024 all over Nigeria.
The Dashboard was deployed on Streamlit and can be accessed via the link below:
https://rainfall-dashboard-for-nigeria-2014-to-2024-fpqyuxwpf2bq9hmzwq.streamlit.app/
